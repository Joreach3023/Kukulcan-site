Support Kukulcan

Si vous rencontrez un problème avec Kukulcan, commencez par vérifier les étapes ci-dessous. Elles couvrent les cas les plus fréquents avant de contacter le support.

1. Redémarrer l’application

Fermez complètement Kukulcan, puis ouvrez l’application à nouveau. Cela règle souvent les problèmes temporaires d’affichage ou de session.

2. Vérifier la version

Assurez-vous d’utiliser la dernière version disponible depuis TestFlight ou l’App Store.

3. Décrire le problème

Si le problème persiste, préparez le modèle d’appareil, la version d’iOS, la version de l’application et les étapes pour reproduire le bug.

Contact

Envoyez votre demande à support@kukulcan.app avec une description précise du problème.
